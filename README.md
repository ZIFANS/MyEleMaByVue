# 饿了么 Web App
使用vue cli3构建的SPA移动端网页。

## 技术栈

Vue + Vuex + Vue Router + axios + Mint UI

# 项目运行

推荐用cnpm intall (淘宝镜像，下载包更快)。

```$xslt
git clone https://github.com/ZIFANS/myelemabyvue.git

cd myelemabyvue

npm install

npm run serve (开发编译)

npm run build (打包发布)
```
